# Practica Caso 2 - Gestión de Usuarios en Plataforma de Cursos Online

## 🚀 Quick Start

### Requirements
- Java 17+
- Maven 3.8+
- MySQL 8.0+

### Setup
```bash
# 1. Clone
git clone https://github.com/Cordero-Jose/Practica_Caso2.git
cd Practica_Caso2

# 2. Create Database
mysql -u root -e "CREATE DATABASE practica_caso2"

# 3. Build
mvn clean install

# 4. Run
mvn spring-boot:run

# 5. Visit http://localhost:8080
